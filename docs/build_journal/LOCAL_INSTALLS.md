
# LOCAL_INSTALLS.md

## Local Tools Installed or Upgraded

### Installed via Homebrew (`brew install`)
- `gh` (GitHub CLI) for repository and PR management.
- `terraform` for infrastructure as code.
- `pre-commit` for managing git hooks.
- `tflint` for Terraform linting.
- `checkov` for Terraform security scanning.
- `gitleaks` for detecting secrets in code.

### Installed Outside of Homebrew
- None explicitly mentioned so far.
